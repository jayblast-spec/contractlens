## ✨ Features

- 🚩 **Red flag detection** — automatically surfaces one-sided, abusive, or unusual clauses
- ❌ **Missing clause alerts** — flags key protections that are absent (IP ownership, limitation of liability, etc.)
- 🔎 **Plain-English summaries** — every risk explained in human language, not legalese
- 📊 **Risk score** — overall contract health rating across key categories
- 📝 **Paste or upload** — supports raw text paste and PDF uploads
- 💼 **Common contract types** — employment, SaaS, freelance, NDA, partnership agreements
- ⚡ **Results in seconds** — no waiting, no registration, no per-document fees

## 🚀 Quick Start

```bash
git clone https://github.com/jayblast-spec/contractlens.git
cd contractlens
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
